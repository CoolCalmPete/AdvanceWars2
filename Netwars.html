Netwars

 
 

Netwars is a clone of the Nintendo/Intelligent Systems game “Advance Wars”. It’s a turn-based strategy game.


It features much aspects of the original game, it even has the same units.


Netwars is not ready for a release, but you can download a preview for Windows, Mac OS X and Linux too.


If you want to work on Netwars, contact me or just start working! It’s open source so everyone who wants can work on it.
